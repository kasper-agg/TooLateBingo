# TooLateBingo
Does your co-worker turn up late to work every now and then? Don't worry and play some bingo!

This PHP script lets you create random bingo cards.

## Parameters:
 -p (players: 1)
 -w (width of the square: 5)
 -s (starting number: 1)
 -m (max numbers: 99)
